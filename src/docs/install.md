# 安装

可以选择下面三种包管理器的任意一种，推荐pnpm。

## npm


```Shell
    npm install tracer-ui
```

## yarn

```Shell
    yarn add tracer-ui
```

## pnpm

```Shell
    pnpm install tracer-ui
```
